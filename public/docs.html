Public API Docs

Cosmic Archive API

A small sci-fi API for sectors, anomalies, and custom mission briefs. No auth required.

Base URL:

GET/api/health

Quick uptime and metadata check.

curl /api/health

GET/api/sectors

Returns all known sectors. Optional filter: ?dangerLevel=high.

curl "/api/sectors?dangerLevel=high"

GET/api/sectors/:id

Returns one sector plus related anomalies and mission count.

curl /api/sectors/umbra-veil

GET/api/anomalies

List anomalies. Filters: ?sectorId=..., ?type=signal, ?minSeverity=4.

curl "/api/anomalies?type=signal&minSeverity=4"

GET/api/feed

Returns a mixed exploration feed of anomalies and open missions. Optional ?count=5.

curl "/api/feed?count=5"

POST/api/missions

Create a mission brief. Required JSON fields: captain, sectorId, objective.

curl -X POST /api/missions \
  -H "Content-Type: application/json" \
  -d '{
    "captain": "Mira Keene",
    "sectorId": "ember-null",
    "objective": "Capture thermal readings inside Ember Mouth",
    "shipClass": "surveyor",
    "riskBudget": 4
  }'

Response Shape Example

{
  "mission": {
    "id": "msn_ab12cd34",
    "callsign": "EMBER-482",
    "captain": "Mira Keene",
    "sectorId": "ember-null",
    "objective": "Capture thermal readings inside Ember Mouth",
    "shipClass": "surveyor",
    "status": "briefing",
    "riskBudget": 4,
    "createdAt": "2026-05-02T21:00:00.000Z"
  }
}

Auth: none. This API is intentionally open for demos and experiments.
